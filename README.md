# 🛍️ Fashion Wear E-Commerce Website

A modern and responsive **multi-page fashion e-commerce website** built using **HTML, CSS, and JavaScript**. This project focuses on delivering a clean design, smooth user experience, and essential shopping features. It is fully deployed and hosted on **Vercel** for fast and reliable performance.

---

## 📌 Features

- 🧥 Multi-page website (Home, Shop, Product, Cart, Contact)
- 📱 Fully responsive design (Mobile, Tablet, Desktop)
- 🎨 Modern and elegant UI/UX
- 🛒 Add to cart functionality
- 🔍 Product browsing and filtering
- 🌙 Dark mode support (if implemented)
- ⚡ Optimized performance and fast loading
- 🌐 Deployed on Vercel

---

## 🛠️ Tech Stack

- **HTML5** – Page structure
- **CSS3** – Styling, layout, and responsiveness
- **JavaScript (Vanilla JS)** – Interactivity and dynamic features
- **Vercel** – Hosting and deployment

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/jossy21-coder/multi-page-Ecommerce-website_using_HTML-CSS-JS.git

2. Navigate into the project directory:
   ```bash
   cd fashion-ecommerce

3. Open the project in your browser:

- Double-click index.html, or
- Use Live Server in VS Code

---

## 🚀 Deployment (Vercel)

This project is deployed using Vercel.

Steps to deploy your own version:
1. Push your project to GitHub
2. Go to https://vercel.com
3. Sign in with GitHub
4. Click "Add New Project"
5. Import your repository
6. Click Deploy
7. Your site will be live instantly 🎉
